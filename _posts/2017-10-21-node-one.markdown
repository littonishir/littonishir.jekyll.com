# 2.Node.js创建项目
安装koa网络框架
```java
npm install koa-generator -g
```
创建项目 koa2 [项目名]
```java
koa2 ishir
```
进到项目目录
```java
cd ishir
```
为项目添加依赖
```java
npm install --save
```
